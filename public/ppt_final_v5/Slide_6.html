2.1 핵심 기능 매트릭스

기능 명칭 핵심 내용 주요 기술 / 특징
디지털 사전 문진 환자가 모바일/웹을 통해 통증, 기능 제한, 관련 증상 등을 체계적으로 입력 표준화 척도 (VAS, DN4, ODI, Red Flags) 직관적 UI/UX 반응형 디자인
AI 기반 분석 리포트 입력된 데이터를 AI 알고리즘으로 분석하여 위험도, 예상 원인 등을 시각화/점수화 머신러닝 패턴 분석 데이터 시각화 엔진 위험도 스코어링
환자 맞춤형 리포트 분석 결과를 환자가 이해하기 쉬운 그래프, 요약 메시지, 행동 제안 형태로 제공 개인화 콘텐츠 생성 이해도 중심 디자인 행동 유도 (CTA)
병원 시스템 연동 환자 리포트 결과를 EMR(전자의무기록) 시스템과 안전하게 연동하여 의료진에게 전달 API 연동 데이터 보안 프로토콜 HL7/FHIR 고려 HIPAA 준수
AI 챗봇 ‘척추 톡톡’ 생성된 리포트 내용을 기반으로 환자의 추가 질문에 실시간 답변 및 정보 제공 RAG (검색 증강 생성) 자연어 처리 (NLP) 맥락 인지 상담
(부가) OCR 연동 디지털 기기 사용이 어려운 환자를 위한 종이 문진표 자동 데이터 변환 지원 광학 문자 인식 (OCR) 접근성 향상
